Roll procedure:

  1. Roll a d20. This die is called the target die.
  2. Add up the values of the current attribute and skill. Then add any situational modifications. This number is called the modifier
  3. Roll a second d20. This die is called the task die
  4. Use the modifier and use it to modify the task die by adding of subtracting up to the value of the modifier.
    • If you can, during the previous stage match the value of the target die you succeed. Any unused modifier value left over counts as the level of succeeses.
    • if you can not manage to match the value. The task counts as a fail.
  5. If you happen to roll a matching set of twentes or ones on the target and task dice. The task counts as a fantastic success with the level of it being the value of the modifier.
  6. If the dice happen to be a one and a twenty, the task counts as a spectacular failure.

Damage procedure:

  1. Roll the damage die. (1d6 for one handed weapons and 2d6 for two handed weapons.)
  2. Add the number off successes from the roll to attack
  3. Subtract any possible armor value. Then subtract the targets toughness value.

Initiative procedure

  1. Make a roll using the Wits or Nimbleness attribute and the weapon skill you plan to use. This only needs to be done at the start of the combat
  2. The proceed to have the combatants state their offencive and defencive action for the turnstarting with the lowest number of successes and continue in accending order until all combatants have declared their actions for the turn
  3. Then in the reverse order, from most successes to the lowest, resolve those actions
